# Vet Clinic Website — UI/UX Design

An award-winning UI/UX design project for a veterinary clinic website, created 
in Figma. The design focuses on a clean, trustworthy visual style appropriate 
for a healthcare/pet-care brand, with intuitive navigation and a calming color 
palette.
